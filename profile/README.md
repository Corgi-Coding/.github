## 技术架构

当前柯基编码的技术架构主要围绕 `typescript` 语言选择

- 前端使用 `vue3.x`
- 后端使用 `nest.js`
- 数据库使用 `pgsql + prisma(orm)`
- 测试工具为 `vitest` + `cypress`
- 自动部署利用 `github actions`
